Data Set and Model

The app uses a machine learning model trained on MIMIC-III data set. The Natual Language Processing model trained in this app is comprised of two parts: a Gradient Boosted Machine model using n-grams of ICD-9 code description reducing the log loss, then a one layer Neural Network learning the probability residuals using internal word embedding.

Why so few ICD-9 codes?

Due to computation power limitation, only four ICD-9 codes are trained so far. However, this app is good enough for a proof of concept.

Future Plans

The combined probability of the two steps does not necessarily fall in [0, 1]. To solve this, a threshold could be learned for the final classification. It will also control the recall and precision of the prediciton of a certain ICD-9 code.
